# tailor-nano Quickstart (1 minute)

## Install

```bash
cd your-project
node /path/to/tailor-nano/install.js
```

Restart Claude Code.

## Use

Just work normally. The system:
- Adds 6 research-backed patterns to `CLAUDE.md`
- Captures errors silently
- Tracks patterns automatically

## Check Status

```
/nano-status
```

Shows error patterns from your work sessions.

## That's It

No configuration. No maintenance. Just proven patterns + memory.
